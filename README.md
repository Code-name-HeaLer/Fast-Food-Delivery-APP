## <a name="installation">ğŸ“± Installation Guide</a>

### Detailed Installation Steps

**1. Clone and Navigate**
```bash
git clone https://github.com/your-username/quickbite-food-delivery.git
cd quickbite-food-delivery
```

**2. Install Dependencies**
```bash
# Using npm
npm install

# Or using yarn (if preferred)
yarn install
```

**3. Environment Configuration**
Create a `.env` file in your project root:

```env
# Appwrite Configuration
EXPO_PUBLIC_APPWRITE_PROJECT_ID=your_project_id
EXPO_PUBLIC_APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
EXPO_PUBLIC_APPWRITE_DATABASE_ID=your_database_id
EXPO_PUBLIC_APPWRITE_USER_COLLECTION_ID=your_users_collection
EXPO_PUBLIC_APPWRITE_FOOD_COLLECTION_ID=your_food_collection
EXPO_PUBLIC_APPWRITE_ORDERS_COLLECTION_ID=your_orders_collection

# Google Services
EXPO_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_maps_key
EXPO_PUBLIC_GOOGLE_OAUTH_CLIENT_ID=your_google_oauth_client

# Payment Processing
EXPO_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_public_key

# Sentry Configuration
SENTRY_DSN=your_sentry_dsn

# Development
NODE_ENV=development
```

**4. Platform-Specific Setup**

**iOS Setup:**
```bash
# Install CocoaPods (if not already installed)
sudo gem install cocoapods

# Navigate to iOS directory and install pods
cd ios && pod install && cd ..
```

**Android Setup:**
- Ensure Android Studio is installed
- Set up Android SDK and emulator
- Configure environment variables for Android SDK

**5. Start Development**
```bash
# Start the Expo development server
npx expo start

# For specific platforms
npx expo start --ios        # iOS only
npx expo start --android    # Android only
npx expo start --web        # Web version
```

## <a name="configuration">ğŸ”§ Configuration</a>

### Appwrite Backend Setup

**1. Create Appwrite Account**
- Sign up at [Appwrite Cloud](https://cloud.appwrite.io/)
- Create a new project

**2. Database Configuration**
Create the following collections:

```javascript
// Users Collection
{
  id: "users",
  attributes: [
    { key: "email", type: "string", required: true },
    { key: "name", type: "string", required: true },
    { key: "avatar", type: "string", required: false },
    { key: "phone", type: "string", required: false },
    { key: "addresses", type: "string[]", required: false }
  ]
}

// Food Items Collection
{
  id: "foods",
  attributes: [
    { key: "name", type: "string", required: true },
    { key: "description", type: "string", required: true },
    { key: "price", type: "double", required: true },
    { key: "category", type: "string", required: true },
    { key: "image", type: "string", required: true },
    { key: "restaurant", type: "string", required: true },
    { key: "rating", type: "double", required: false },
    { key: "ingredients", type: "string[]", required: false }
  ]
}

// Orders Collection
{
  id: "orders",
  attributes: [
    { key: "userId", type: "string", required: true },
    { key: "items", type: "string", required: true }, // JSON string
    { key: "total", type: "double", required: true },
    { key: "status", type: "string", required: true },
    { key: "deliveryAddress", type: "string", required: true },
    { key: "createdAt", type: "datetime", required: true }
  ]
}
```

**3. Authentication Setup**
- Enable Google OAuth in Appwrite Console
- Configure OAuth redirect URLs
- Set up session management

### Google Services Configuration

**1. Google Maps API**
- Enable Maps SDK for iOS and Android
- Enable Places API
- Get API key from Google Cloud Console

**2. Google OAuth**
- Create OAuth 2.0 credentials
- Configure consent screen
- Add authorized redirect URIs

### Payment Integration

**Stripe Setup:**
```bash
# Install Stripe React Native SDK
npm install @stripe/stripe-react-native
```

Configure in your app:
```typescript
import { StripeProvider } from '@stripe/stripe-react-native';

export default function App() {
  return (
    <StripeProvider publishableKey="your-publishable-key">
      {/* Your app components */}
    </StripeProvider>
  );
}
```
